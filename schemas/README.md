# Schemas Directory

Schemas to be used by the data-definition-engine. These schema files reflect the latest versions. Updates to the schemas should not be made in this repo.

## Directory Structure

```structure
schemas/
├── dds/                        # LinkML schema from https://github.com/cdisc-org/DataExchange-DDS/model (define.yaml)
├── define/                     
│   ├── 2.0/                    # Define-XML 2.0 schema (define2-0-0.xsd)
│   ├── 2.1/                    # Define-XML 2.1.11 schema (define2-1-0.xsd)
└── odm/
│   ├── 1.3.2/                  # ODM 1.3.2 schema (ODM1-3-2.xsd)
│   ├── 2.0/                    # ODM 2.0 schema (ODM.xsd)
```
